# for 循环

```js
println("demo1 ================")
for a = 0; a < 5; a++ {
    println("current value:", a)
}

println("demo2 ================")
arr = [1, false, true, 3.14, "changlie"]
for a = 0 ; a < arr.size(); a++ {
    println("current value:", arr[a])
}
```

Quick的for循环与golang的很像。
